# Java Student Grade Tracker

A simple and interactive Student Grade Tracker web application developed using HTML, CSS, and JavaScript. This project helps users manage student records, calculate grades, track academic performance, and view results in an organized manner.

## 📌 Features

* Add student details
* Enter and manage subject marks
* Automatic grade calculation
* Percentage calculation
* Performance tracking
* User-friendly interface
* Responsive design
* Real-time updates using JavaScript

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)


## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/Thol-444/CodeAlpha_tasks-java-student-grade-tracker.git
```

2. Navigate to the project folder:

```bash
cd CodeAlpha_tasks-java-student-grade-tracker
```

3. Open `index.html` in your browser.

## 🎯 Project Objective

The objective of this project is to provide a simple grade management system that allows users to store student marks, calculate percentages, determine grades, and monitor academic performance efficiently.
